# Security Policy

## Supported Versions

The following versions of `indianconstitution` are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.1.0  | :white_check_mark: |
| < v1.1.0| :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please do not report it through public issues. Instead, please report it via the following process:

1. **Email the maintainer**: Send an email to **vikhrams@saveetha.ac.in**.
2. **Details**: Include as much information as possible, including steps to reproduce, potential impact, and any suggested fixes.
3. **Response**: You can expect an acknowledgment within 48-72 hours.
4. **Resolution**: We will work with you to resolve the issue and coordinate a public disclosure if necessary.

Thank you for helping keep this project secure!
